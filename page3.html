Reducing Impact on the Environment

How can we minimize the impact of computer device production on the environment?

  • Change:We can minimize the impact in many different ways. One way to do it is to use resources more efficiently. We can try to make computers using better design tactics to reduce the materials used.
  • Use less:Another way is to only purchase the essential things you need instead of buying more unnecessary things.
  • Re-use reduce recycle:Another way we can help is to recycle and reuse. When we are done with the components we should recycle them properly so they don't end up in the wrong places hurting the environment.
